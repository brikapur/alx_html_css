# CSS Advanced

This project builds upon the **HTML Advanced** project by introducing advanced styling concepts using CSS.  
The goal is to recreate the full **SmileSchool** webpage layout as provided in Figma, while following best practices in modern CSS.

##  Project Description

In this project, I focused on converting a fully structured HTML layout into a visually appealing website using CSS.  
The design specification was provided through Figma, and my task was to implement:

- Responsive layout
- Flexbox-based structure
- Advanced styling techniques
- Proper typography and spacing
- Integration of icons and images
- Accurate recreation of the designer’s vision

## Technologies Used

- **HTML5**
- **CSS3**
- **Flexbox**
- **FontAwesome**
- **Google Fonts**
- **Responsive Web Design

##  How to Run the Project

1. Clone the repository:
   git clone https://github.com/brikapur/alx_html_css.git
2. Navigate to the project folder:
   cd alx_html_css/css_advanced
3. Open the page in a browser:
   - Double-click `index.html`
   - Or use VS Code Live Server
