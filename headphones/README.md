Headphones Website

This project is part of the ALX HTML & CSS course. The goal is to recreate a webpage from a Figma design using HTML and CSS.

Figma

Open the Figma file given in the project.

Click “Duplicate to your Drafts” so you can inspect the design.

Install the fonts used (Source Sans Pro and Spin Cycle OT) if needed.

Requirements

Page must switch to mobile view at 480px or smaller.

Links hover/active color: #FF6565

Buttons hover/active: opacity 0.9

Maximum content width: 1000px and centered.
